PROJECTS



GOODBOWL

GoodBowl uses data from the Edamam API to display a list of buddha bowl recipes with nutritional information for each recipe. I used React for the front-end as this meant I could re-use components throughout the site, making it easier to maintain and refactor. I used Express for the back-end, as I find it's the easiest and simplest way to get a server up and running for small projects like this.

VISIT GOODBOWL

RESTAURANT ROULETTE

Restaurant Roulette is a React app with an Express back-end which obtains the users location, then provides details of a random restaurant in the vicinity to remove the analysis paralysis usually involved when choosing somewhere to eat! Data is pulled from the Zomato API.

VISIT RESTAURANT ROULETTE

PASTEL-FLAVOURED WEATHER APP

I made this as part of the freeCodeCamp curriculum using JavaScript. The app shows local weather based on the users location which is obtained using the HTML5 geolocation API.

VISIT WEATHER APP

INSTAGRAM GALLERY

I was interested in learning more about Vue, so built this site which displays Instagram posts as a gallery view to learn about the core concepts. I plan to expand this to a site which allows users to enter their own login details and create their own gallery linked to their account.

VISIT INSTAGRAM GALLERY
